# 2D Game Engine

Built with TS and WebGL2.

# Dependencies
This engine relies on these to run a Prod build:
 * gl-matrix
 * lodash

There are some dev dependecies, see [package.json](https://github.com/IgorSteps/nucleo/blob/master/package.json) for them.

To install, run 
```
npm i
```

# Running locally
Compile 
```
npm run build
```
Serve 
```
npm run develop
```
